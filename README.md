# Cybersecurity Anomaly Detection Dashboard

## Project Overview
An interactive Tableau dashboard designed to identify global network threats. 
This project correlates traffic volume with geographic origin to detect 
suspicious activity.

## Key Features
- **Geographic Mapping:** Visualizes traffic by origin using Latitude/Longitude.
- **Anomaly Detection:** Color-coded labels (Normal vs. Suspicious).
- **Volume Analysis:** Proportional sizing based on `Bytes In`.
